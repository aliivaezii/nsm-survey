# NSM Survey

Academic survey tool for the research paper:
**Selecting the North Star: A Mixed-Methods Decision Framework for Growth Hacking Metric Selection**

## Authors
- Arman Karimian — University of Turin (researcher)
- Ali Vaezi — Politecnico di Torino (co-researcher, developer)
- Prof. Gabriele Santoro — University of Turin (supervisor)

## Stack
Static HTML · Supabase · Vercel

## Setup
1. Open `index.html` and find the `CONFIG` block near the top of the script
2. Set `SUPABASE_URL` and `SUPABASE_ANON_KEY` from your Supabase project dashboard
3. Run the SQL in the README comment block inside `index.html` to create the response table
4. Push to GitHub and connect to Vercel — deploy with default settings
